## Tipos

--

¿Qué es un tipo?

Un tipo es un conjunto de características que identifican el comportamiento de un valor en particular.

--

## División de los tipos

--
**Primitivos:**

* String
* Number
* Boolean
* Symbol
* null
* undefined

--
**Objetos (Por referencia)**

* Function
* Array
* Object
* Maps
* Set
* Weakset
* WeakMap
* Date
* RegExp
* Error
